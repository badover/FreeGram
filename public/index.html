ABOUT FREEGRAM

ABOUT THIS SITE

Freegram is an open-source messenger where users can create or join rooms for chat, voice calls, and file sharing.

WHAT IS A ROOM

A room is a private chat space. People with the same room name and key can join and communicate together.

FGRAM CHAT

Use fgram chat to send quick messages to everyone in the room.

VOICE CALLS

Join a voice call in the room for live conversation. You can mute your microphone or deafen audio at any time.

IMAGE SHARING

Send images and files in the room so participants can view shared content in one place.
